UCL QuizApp - User guide

Content Guide

The following guide explains how to install and use the UCL QuizApp, a location-based mobile app. Here is the content of the guide:

-        System requirements.

-        How to install the app.

-        How the app works (functionalities of both the left-side and the top-right menu).

-        How to play.

System requirements

The app works on the Android OS, while the iOS and the Windows OS are not supported. The minimum supported version of Android is the 5.1.0 (further information about the Android versions supported on the Phonegap website).

The app must to be connected to the UCL VPN to load the points of interest. We recommend the Cisco AnyConnect app to establish the VPN connection; further information about how to connect to the UCL VPN can be found on the following link.

How to install the app

Since the app is not distributed through the Google Play Store, the user must allow the installation from unknown sources on his device, otherwise the app cannot be installed. Usually, this option is under the security section within the settings.

The installation process needs to be initialized from the following link: https://build.phonegap.com/apps/3148274/share.

Once here the user has two possibilities:

1-     Through its own mobile phone by scanning the QR code provided and following the instructions supplied by the device to complete the process;

2-     By downloading the .apk file on the PC and then connecting the phone to the computer to complete the installation on the mobile device.

How the app works

This is a location-based app that uses the capability of mobile devices to track the user position in order to visualise questions related to some points of interest (mainly related to the UCL campus). The app is composed of three main parts: the main body, the left-side menu and the top-right menu.

The main body

The main body contains the visualisation tools that allow the user to get the questions and answer to the quiz. This part consist of:

-        The map: here the user can see his position and the location of the interest points. each point is associated to a question to be answered.

-        The coordinates of the tracked position: this option let the user to see the coordinates of his position expressed in decimal degrees (further information about the coordinates can be found here).

-        The question: this section contains the question and the related answers that appear when the user is near enough to an interest point.

-        The check answer button: once the user selects his answer can check if it is correct by clicking this button.

The left-side menu

This menu can be opened by tapping the top-left three-lines icon on the interface. Here the user has four different options:

-        Load Points: the button allows to load the points of interest. Before to start the game the points need to be loaded, or else there are no question to answer.

-        Track Location: once clicked, the option activate the tracking and search for near points. If the user is nearby to one of them a popup will appear and the question will be loaded, otherwise a popup will notify the user that he is far from any point of interest.

-        Get Questions: the user can check if there is a point near enough by clicking this option. If yes, a popup will appear and the question will be loaded; if not a popup will notify the user that he is far from any point of interest.

-        Clear View: this option clean the map. The user should use this option just in case he wants to restart the game.

The top-right menu

From the top-right menu the user can access both to this user guide (User Guide link) and to additional documentation that gives more information about the app and its functionalities (More Info link).

How to play

Before to start the app, the user must be connected on the UCL VPN. Once the connection is established, the user can open the app and taps on Load Points, to load all the points of interest. After that, the user should tap on Track Location to know if he is nearby to a point; if yes, the app will display the related question, otherwise he needs to go towards a point and taps on Get Questions to visualise the question and answer the quiz.
